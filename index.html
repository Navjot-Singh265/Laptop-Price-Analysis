📐 Regression 1 Equation

Price = 557.40 + 67.70 × RAM
For every 1GB increase in RAM, laptop price rises by ~€67.70. R² = 0.55 means RAM explains 55% of price variance.

📐 Regression 2 Equation

Price = -565.8 + 0.866 × ScreenW
Screen width explains 31% of price variance. Premium 4K displays (3840px wide) command significantly higher prices.

🏆 Best Model Wins

RAM → Price is clearly the better model (R²=0.55 vs 0.31). RAM is the top single predictor for laptop pricing in this dataset.

🔮 Future Improvement

Multiple regression using RAM + ScreenW + CPU_freq together should push R² above 0.70. Try Ridge/Lasso for regularization.

T-Test: Price Distribution — High RAM vs Low RAM
H₀: μ(High RAM) = μ(Low RAM) | H₁: μ(High RAM) > μ(Low RAM) | α = 0.05
p ≈ 0.000 ✅

📋 Null Hypothesis (H₀)

Laptops with HIGH RAM (≥16GB) have the SAME average price as LOW RAM laptops (<16GB). No significant difference exists.

📋 Alternative Hypothesis (H₁)

Laptops with HIGH RAM (≥16GB) have a SIGNIFICANTLY HIGHER average price. One-tailed test with α = 0.05.

✅ Result: Reject H₀

p-value ≈ 0.000 << 0.05. We REJECT the null hypothesis. High RAM laptops (μ≈€1,848) cost significantly more than Low RAM laptops (μ≈€886).

💡 Business Insight

RAM is a major premium driver. Manufacturers charge ~€962 more on average for ≥16GB RAM configs. This guides consumer buying decisions significantly.
