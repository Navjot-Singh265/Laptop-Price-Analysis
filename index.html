📋 H₀

Laptops with RAM ≤ 8GB have the SAME average price as RAM > 8GB. No significant difference.

📋 H₁

Significant price difference between the two groups. Two-tailed test using ttest_ind().

✅ Conclusion

p=0.003 < 0.05 → Reject H₀. High RAM (>8GB) avg €1,441 vs €1,030 — a €411 difference.
