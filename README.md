# 💻 Laptop Price Analysis Hello navjot

## 📊 Project Overview
This project analyzes laptop prices based on hardware specifications such as **RAM, CPU, storage, and company** using Data Science techniques. It combines **Exploratory Data Analysis (EDA), visualization, regression modeling, and hypothesis testing** to derive meaningful insights.


---

## 🎯 Objectives

1. **RAM vs Price Analysis**
   - Analyze relationship between RAM and price  
   - Apply **Simple Linear Regression**  
   - Predict price for given RAM (e.g., 8GB)  

2. **Laptop Type Distribution**
   - Visualize proportion of laptop types using **Pie Chart**  

3. **RAM Configuration Analysis**
   - Identify most common RAM configurations using **Bar Graph**  

4. **Company-wise Analysis**
   - Analyze number of laptops produced by each company  

5. **Hypothesis Testing (T-Test)**
   - Compare prices of low RAM (≤ 8GB) and high RAM (> 8GB) laptops  
   - Check statistical significance

## 📉 Model Used

### 🔹 Simple Linear Regression

Equation:

Price = m × RAM + c

Used to:
- Understand impact of RAM on price  
- Predict price for new values  

---

## 🧪 Hypothesis Testing

- **Test Used:** Independent T-Test  
- **Purpose:** Compare price between two RAM groups  

### Hypothesis:
- H₀: No significant difference in price  
- H₁: Significant difference exists  

---

## 📌 Key Insights

- 📈 Higher RAM leads to higher laptop prices  
- 💻 8GB and 16GB are the most common configurations  
- 🏢 Some companies dominate the dataset  
- 📊 Statistical testing confirms RAM significantly affects price
