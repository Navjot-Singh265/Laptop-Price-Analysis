# 💻 Laptop Price Analysis

## 📊 Project Overview
This project analyzes laptop prices based on hardware features like RAM, CPU, and storage using Data Science techniques.

## 🎯 Objectives
- RAM vs Price Analysis
- CPU vs Price Analysis
- Storage vs Price Analysis
- Hypothesis Testing using T-test

## 🛠️ Tools & Technologies
- Python
- Pandas
- Seaborn
- Matplotlib
- Scikit-learn

## 📈 Techniques Used
- Exploratory Data Analysis (EDA)
- Linear Regression
- Correlation Analysis
- Hypothesis Testing

## 📊 Key Insights
- RAM significantly impacts price
- CPU performance affects cost
- Storage plays a role in pricing

## 📌 Conclusion
Laptop pricing is strongly influenced by hardware specifications like RAM, CPU, and storage.
